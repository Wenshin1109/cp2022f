Password strength

1. N 個英文字母小寫加 N 分。
2. X 個英文字母大寫加 X*3 分。
3. Y 個數字加 Y*2 分。
4. M 個特殊符號{ ~!@#$%^&*<>_+=}加 M*4.5 分。
5. 有五個以上的數字且任兩個數字在密碼中的位置不相鄰， 例如：「1a1a1a1a1」，加 10 分。

輸入說明：

輸入 N(2<=N<=5)組密碼，每組以換行區隔，且任兩組密碼 strength 不會相同，可一直輸入，直到輸入-1 停止。

輸出說明：

計算各組密碼 strength，依序輸出最高及最低兩組密碼及其 strength，兩組密碼以換行區隔，密碼與分數間以空格區隔。

Input/Output

hints


            #include <stdio.h>
            #include <string.h>
            int main(){
              // 輸入字串
              // 如果該字串為-1，停止輸入，開始比較強度並輸出最強&最弱
              // 如果該字串為密碼，計算該密碼之強度
              // 注意：密碼強度是浮點數
              // 特殊符號包含：{ ~!@#$%^&*<>_+=}
              return 0;
            }

文字編輯器

設計一個命令列簡易文字編輯器，大小寫視為不同。

輸入說明：

1. 第一行輸入一個整數 M, N， M 是原始文章行數，N 是編輯指令數。
2. 其後 M 行，每一行為原始句子，以空白隔開每個字。
3. 其後 N 行，每一行的第一個字串為編輯指令，隨後為對應的控制字串，以空白隔開。
4. 文章不含標點符號；第幾行、第幾個字都是從 1 開始計算。

輸出說明：

請輸出編輯完成之文章。

Input/Output

hints


          #include <stdio.h>
          #include <string.h>
          int main(){
            int m;  // 原始文章行數
            int n;  // 指令數
            for(int i = 0; i < m; i++){
              // 輸入原始文章
            }
            for(int i = 0; i < n; i++){
              // 輸入指令
              // 利用基礎題03的substring comparison去判斷指令
              // 常用string函式：strcpy()、strcat()、strchr()
            }
            return 0;
          }
